# atm-ui
- Para iniciar este projeto é necessario ter o **node** e o **bower** instalados e configurados. Digite esses comandos **NESSA ORDEM** para iniciar a aplicação:
	- npm install
	- bower install
	- npm start

Dados para acessar a aplicação: *account:* 123, *password:* 123
